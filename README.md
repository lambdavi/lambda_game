## Lambda Game
# Requirements: 
pygame | pytmx | python (3.7+)
```
$ pip3 install pygame
```
```
$ pip3 install pytmx
```
# Installation
First of all clone the repo.
Then navigate to code section:
```
$ cd lambda_game/code/
```
and execute the code:
```
$ python3.7 main.py
```
